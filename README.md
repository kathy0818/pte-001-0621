## 1. 这个项目在干什么

把 **APEUni / 猩际 PTE 模考**的"详解"截图（原始 PDF 不可复制、只能截图）**逐题人工誊写、重新排版**，做成干净、可读、可打印的 **错题整理卷 PDF**。

- 当前这套是 **PTE Core 模考 1B**（总分 40：听力 33 / 阅读 42 / 口语 29 / 写作 57）。
- 每个版面完整保留每题的：**题干 / 原文、范文、你的作答、逐项评分、彩色批改、AI 中文建议**。
- 错误按平台标注用颜色还原；平台**漏标**的错误用橙色"补"标记额外补上。
- 成品强调"忠于原图"：内容均为人工对照誊录，若与原图有出入以原图为准。

## 2. 怎么"看"成品（最快路径）

1. **最快**：直接打开 **`整理卷/`** 根目录里的 4 个最终成品 `PTE模考1B_{口语,写作,阅读,听力}_整理卷.pdf`。
2. 想看历史版本：进 `整理卷/<部分>/`（如 `整理卷/听力/`），里面是该 part 的全部版本，文件名形如 `001_PTE模考1B_听力_整理卷.pdf`，**序号最大的就是最新版**。
3. 想自己复现/改：对那一版的 `.tex` 跑一次 XeLaTeX 即可（见 §5）。纯文字版本的 `.tex` 都是**单文件自包含**、不依赖图片。

## 4. 成品版本规则（重要 —— 必须遵守）

用户定的规矩，新会话务必照做：

1. **一个版本 = 一对成品**：`<序号>_PTE模考<编号>_<部分>_整理卷` 的 `.pdf` + `.tex`。
2. **版本化存档按 part 分文件夹**：`整理卷/{口语,写作,阅读,听力}/`，每个 part 的所有版本进自己的子文件夹（`flatten.py` 已直接输出到这里）。
3. **根目录只放 4 个最终成品 PDF**：每 part 最新版另复制一份到 `整理卷/` 根，改干净名 `PTE模考<编号>_<部分>_整理卷.pdf`（给人直接取用；出新版后更新这份）。
4. **每个 part 各自从 `000` 起编号**（互不干扰），每更新一版 **+1**（三位数）。某 part 最新版 = 它子文件夹里序号最大的。
5. **绝不修改、绝不覆盖任何旧版本**——只**新增**下一个序号。旧的永远保持原样。
6. **纯文字版本**：`.tex` 单文件自包含，放 `整理卷/<部分>/`（写作/阅读/听力即如此）。
   **含图片版本**（如口语 DI 配图）：做成**子文件夹** `整理卷/口语/<序号>_…_整理卷/`（`.tex` + `img/` + `.pdf`）。两者都用 `build/flatten.py` 生成（含图片时多传一个图片目录名，见 §6）。
7. **原始素材**统一在 `原卷素材/`（只读）；**交接文档**统一在 `交接文档/`。

## 5. 环境与编译

**引擎：XeLaTeX**（中英混排，需 CJK 字体）。需要的字体：

- `Noto Sans CJK SC`、`Noto Serif CJK SC`（中文）
- `TeX Gyre Heros`（拉丁，Helvetica 风格）

安装（Debian/Ubuntu 示例）：

```bash
sudo apt-get install -y texlive-xetex texlive-latex-extra \
    fonts-noto-cjk fonts-texgyre
```

**编译某一版成品**（单文件，自包含）：

```bash
cd 整理卷
xelatex 001_PTE模考1B_写作_整理卷.tex      # 跑两遍，tcolorbox/fancyhdr 引用才稳定
xelatex 001_PTE模考1B_写作_整理卷.tex
```

**在工作区预览/继续排版**（编辑后快速看效果）：

```bash
cd build/paper
xelatex preview_writing.tex
xelatex preview_writing.tex
```

## 6. 新会话的标准流程（下次怎么继续）

1. **读 `README.md` + `交接文档/制作流程.md`（+ 对应分册）**，确认环境（`xelatex` + 上面三套字体都在）。
2. **看最新版 PDF**（`整理卷/` 根的成品，或 `整理卷/<部分>/` 里序号最大的）了解版式与配色风格，保持一致。
3. **在 `build/paper/` 工作**：
   - 新部分（如口语）→ 新建 `sec_speaking_body.tex` 和 `preview_speaking.tex`，复用 `ptestyle.sty`；
   - 改写作 → 编辑 `sec_writing_body.tex` / `cover.tex`；
   - 反复 `xelatex preview_*.tex` 预览，放大截图核对原图。
4. **产出新版本**：`python3 build/flatten.py <part英文> <中文> <考试号> <下一个序号> [图片目录]` → 输出到 `整理卷/<部分>/`。无图→单文件 `.tex`；有图（把图放 `build/paper/<图片目录>/`，正文用 `\includegraphics{<图片目录>/x.png}`）→ 生成子文件夹（tex+img+pdf）。再到目标处 `xelatex`（两遍）。**若是最新版**，把 PDF 复制到 `整理卷/` 根改干净名。**不动任何旧版本**（见 §4）。
5. **提交并推送**：commit message 写清楚改了什么；推到指定的开发分支。

## 7. 批改标注图例（来自 `ptestyle.sty`）

| 宏 | 含义 | 颜色 |
|----|------|------|
| `\fix{原}{正}` | 修改 / 拼写 / 缺空格 | 红删除线 + 绿改正 |
| `\del{词}` | 应删除 | 蓝删除线 + "删" |
| `\ins{词}` | 应插入 | 紫色 + "∧插" |
| `\add{原}{正}` | **平台漏标**、本卷补标 | 橙色波浪线 + "补" |
| `\sfull/\spart/\szero` | 评分：满分 / 部分 / 零分 | 绿 / 橙 / 红 |

颜色主色为平台青绿 `cAccent = #16A085`；四项分数条形图配色见 `cover.tex`。
