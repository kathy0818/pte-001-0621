## 1. 这个项目在干什么

把 **APEUni / 猩际 PTE 模考**的"详解"截图（原始 PDF 不可复制、只能截图）**逐题人工誊写、重新排版**，做成干净、可读、可打印的 **错题整理卷 PDF**。

- 当前这套是 **PTE Core 模考 1B**（总分 40：听力 33 / 阅读 42 / 口语 29 / 写作 57）。
- 每个版面完整保留每题的：**题干 / 原文、范文、你的作答、逐项评分、彩色批改、AI 中文建议**。
- 错误按平台标注用颜色还原；平台**漏标**的错误用橙色"补"标记额外补上。
- 成品强调"忠于原图"：内容均为人工对照誊录，若与原图有出入以原图为准。

## 4. 成品版本规则（重要 —— 必须遵守）

用户定的规矩，新会话务必照做：

1. **一个版本 = 一对文件**：`<序号>_…_整理卷.pdf` + `<序号>_…_整理卷.tex`。
2. **每个项目一个子文件夹**，该项目所有版本都放进同一个子文件夹（写作 → `写作_整理卷/`；以后口语 → `口语_整理卷/`，依此类推）。
3. **序号前缀，从 `000` 开始**，每次会话 / 每次更新 **+1**（`000` → `001` → `002` …，三位数）。
4. **绝不修改、绝不覆盖任何旧版本**——只**新增**下一个序号。旧的 `000` 永远保持原样。
5. 成品 `.tex` 必须是**单文件自包含**（内联样式、无外部图片），保证多年后只靠这一个 `.tex` 就能复现同一 PDF。

## 5. 环境与编译

**引擎：XeLaTeX**（中英混排，需 CJK 字体）。需要的字体：

- `Noto Sans CJK SC`、`Noto Serif CJK SC`（中文）
- `TeX Gyre Heros`（拉丁，Helvetica 风格）

安装（Debian/Ubuntu 示例）：

```bash
sudo apt-get install -y texlive-xetex texlive-latex-extra \
    fonts-noto-cjk fonts-texgyre
```

**编译某一版成品**（单文件，自包含）：

```bash
cd 写作_整理卷
xelatex 000_PTE模考1B_写作_整理卷.tex      # 跑两遍，tcolorbox/fancyhdr 引用才稳定
xelatex 000_PTE模考1B_写作_整理卷.tex
```

**在工作区预览/继续排版**（编辑后快速看效果）：

```bash
cd build/paper
xelatex preview_writing.tex
xelatex preview_writing.tex
```

## 7. 新会话的标准流程（下次怎么继续）

1. **读本 README**，确认环境（`xelatex` + 上面三套字体都在）。
2. **看最新版 PDF**（`写作_整理卷/` 里序号最大的）了解版式与配色风格，保持一致。
3. **在 `build/paper/` 工作**：
   - 新部分（如口语）→ 新建 `sec_speaking_body.tex` 和 `preview_speaking.tex`，复用 `ptestyle.sty`；
   - 改写作 → 编辑 `sec_writing_body.tex` / `cover.tex`；
   - 反复 `xelatex preview_*.tex` 预览，对照 `build/extract/*.txt` 和放大截图核对原图。
4. **产出新版本**：把成品展平成**单文件 `.tex`**，连同编出的 PDF，以**下一个序号**写入对应子文件夹（如 `写作_整理卷/001_…`）——**不动任何旧版本**（见 §4）。
   - 展平 = 把 `ptestyle.sty`（去掉 `\ProvidesPackage`/`\endinput`）放进 `\documentclass` 与 `\begin{document}` 之间，再依次 `\input` 的内容内联进来。可参考 `000` 那一版的做法。
5. **提交并推送**：commit message 写清楚改了什么；推到指定的开发分支。

## 8. 批改标注图例（来自 `ptestyle.sty`）

| 宏 | 含义 | 颜色 |
|----|------|------|
| `\fix{原}{正}` | 修改 / 拼写 / 缺空格 | 红删除线 + 绿改正 |
| `\del{词}` | 应删除 | 蓝删除线 + "删" |
| `\ins{词}` | 应插入 | 紫色 + "∧插" |
| `\add{原}{正}` | **平台漏标**、本卷补标 | 橙色波浪线 + "补" |
| `\sfull/\spart/\szero` | 评分：满分 / 部分 / 零分 | 绿 / 橙 / 红 |

颜色主色为平台青绿 `cAccent = #16A085`；四项分数条形图配色见 `cover.tex`。
